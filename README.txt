Welcome to laraFeedback

There are only few steps to install (which i think you already know but i'm going to list down below) to the laraFeedback

1  - composer install
2  - npm install
3  - npm run dev
4  - cp .env-example .env
5  - change DB name if you want to im using mysql
5  - php artisan key:generate
4  - php artisan migrate --seed

Voila you are good to go....
